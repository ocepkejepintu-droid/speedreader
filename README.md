# Speedreader (RSVP Reader)

Version **1.0.0** — see [`VERSION`](VERSION) and [`CHANGELOG.md`](CHANGELOG.md).

Focused speed-reading PWA: RSVP, Phantom, Scroll, and Paragraph modes. EPUB library with optional Google sync. 263 shared book summaries.

- **Live:** https://zipang.id/rsvp/
- **App:** https://zipang.id/rsvp/app/

## Quick start

```bash
npm install
npm test
./serve.sh
```

Open `http://localhost:8080/rsvp/app/`.

## Deploy

```bash
./deploy-vps.sh
```
